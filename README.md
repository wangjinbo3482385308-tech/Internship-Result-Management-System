# Internship Result Management System (COMP1044)

##  Project Overview
This web-based Internship Result Management System is designed for the university to improve the efficiency, transparency, and accuracy of student evaluations during industrial training.The system replaces manual spreadsheets with a secure, role-based platform for Admins and Assessors.

##  Tech Stack
* **Database:** MySQL
* **Frontend:** HTML, CSS, JavaScript 
* **Backend:** PHP (Server-side scripting) 

##  Key Functionalities
* **User Management:** Admin can add, update, and delete student profiles and manage assessor accounts.
* **Internship Management:** Admin assigns students to assessors and records company details.
* **Result Entry & Mark Calculation:** Assessors enter marks for specific criteria, and the system automatically calculates final scores based on fixed weightages.
* **Result Viewing:** Authorized users can search and filter results by Student ID or Name.

##  Assessment Weightages
The system applies the following exact weightages for mark calculation:
* Undertaking Tasks/Projects: **10%** 
* Health and Safety Requirements: **10%** 
* Connectivity & Use of Theoretical Knowledge: **10%** 
* Presentation of the Report: **15%** 
* Clarity of Language and Illustration: **10%** 
* Lifelong Learning Activities: **15%** 
* Project Management: **15%** 
* Time Management: **15%**
